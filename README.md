# 📈 주식 기술적 지표(RSI) 자동 알람 서비스 구축 가이드

본 문서는 서버 비용 0원(완전 무료)을 지향하며, GitHub Actions, Python, Gmail SMTP, 그리고 Streamlit을 활용하여 개인 투자자용 주식 알람 서비스를 구축하는 전체 로드맵을 담고 있습니다.

---

## 1. 서비스 아키텍처 개요
본 서비스는 **서버리스(Serverless)** 구조를 채택하여 유지비가 전혀 들지 않으면서도 안정적인 작동을 보장합니다.



- **Frontend (사용자 인터페이스):** Streamlit (무료 호스팅)
- **Backend (자동화 엔진):** GitHub Actions (월 2,000분 무료)
- **Database (정보 저장):** Supabase (무료 티어 500MB)
- **Notification (알림):** Gmail SMTP (무료 발송)
- **Data Source:** FinanceDataReader (오픈소스 라이브러리)

---

## 2. [1단계] 핵심 로직 구현 (개인용 자동화)

가장 먼저 내 이메일로 알람이 오게 만드는 단계입니다.

### 2.1 주요 기술 스택
- **언어:** Python 3.9+
- **라이브러리:** `FinanceDataReader`, `pandas`, `smtplib`

### 2.2 RSI 계산 수식
상대강도지수(RSI)는 다음 수식을 통해 계산되며, 과매수/과매도 구간을 판단합니다.

$$RSI = 100 - \left[ \frac{100}{1 + \frac{Average\ Gain}{Average\ Loss}} \right]$$

### 2.3 보안 설정 (Gmail & GitHub)
1. **Gmail:** 구글 계정 설정에서 '2단계 인증' 활성화 후 **'앱 비밀번호(16자리)'**를 발급받습니다.
2. **GitHub Secrets:** 저장소 Settings에서 `EMAIL_USER`, `EMAIL_PW`를 각각 환경 변수로 등록하여 보안을 유지합니다.

---

## 3. [2단계] GitHub Actions를 이용한 스케줄링

매시간 자동으로 주가를 체크하고 이메일을 보내는 작업기 설정입니다.

- **파일명:** `.github/workflows/alert.yml`
- **실행 주기:** `cron: '0 0-7 * * 1-5'` (한국 시간 기준 평일 장중 매시 정각)
- **주요 기능:** 환경 변수를 안전하게 불러와 Python 스크립트를 실행합니다.

---

## 4. [3단계] 사용자 웹 페이지 구축 (Streamlit)

사용자가 직접 알람을 설정할 수 있는 웹 인터페이스입니다.

### 4.1 UX/UI 설계
- **로딩 바 구현:** `st.spinner` 또는 `st.progress`를 사용하여 서버가 깨어날 때 사용자에게 작업 진행 상황을 시각적으로 알립니다.
- **슬립 모드:** Streamlit 무료 티어는 7일간 접속이 없으면 잠들지만, 사용자가 접속하면 10~20초 내에 자동으로 다시 깨어납니다.

### 4.2 데이터베이스 연동 (Supabase)
- 사용자의 이메일, 관심 종목, 타겟 RSI 값을 테이블에 저장합니다.
- **보안(RLS):** Row Level Security를 설정하여 사용자가 자신의 설정 정보만 조회/수정할 수 있도록 보호합니다.
